# Datahen

Welcome to the DataHen gem, this gem includes the client and the CLI to be able to integrate with datahen.com

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'datahen'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install datahen
